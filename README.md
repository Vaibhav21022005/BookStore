#  Urban Bean Café – Responsive Café Website

Urban Bean Café is a modern, responsive, and visually appealing café website built using **HTML, CSS, and JavaScript**.  
The project focuses on clean UI, smooth user experience, and mobile-first responsiveness.

---

## 🌐 Live Preview
👉 (https://precious-gnome-10a0d6.netlify.app/)

---

## 📌 Project Features

- ✅ Fully responsive design (Desktop, Tablet & Mobile)
- ✅ Modern café-themed UI & color palette
- ✅ Mobile hamburger menu with smooth toggle
- ✅ Swiper.js based Testimonials slider
- ✅ Organized multi-page structure
- ✅ Clean and reusable CSS
- ✅ Hover effects and smooth transitions
- ✅ Well-structured and readable code

---

## 📂 Pages Included

- **Home (`index.html`)**
  - Hero section
  - Call-to-action buttons
  - Navigation bar & footer

- **About (`about.html`)**
  - Café introduction
  - Brand story
  - Image and content layout

- **Menu (`menu.html`)**
  - Special Menu
  - Coffee
  - Tea
  - Snacks
  - Chinese
  - Pizza (6 varieties)
  - Prices with categories

- **Testimonials (`testimonials.html`)**
  - Swiper slider
  - Customer reviews
  - Navigation arrows & pagination dots

- **Gallery (`gallery.html`)**
  - Café images
  - Hover zoom effects
  - Grid-based layout

- **Contact (`contact.html`)**
  - Contact information
  - Contact form
  - Address, email & phone details

---

## 🛠️ Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling & Responsive Design
- **JavaScript (ES6)** – Interactivity
- **Swiper.js** – Testimonials slider
- **Font Awesome** – Icons
- **Google Fonts** – Typography
